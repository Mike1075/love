# 媒体文件目录

这个目录用于存放幻灯片的视频和图片文件。

## 📁 文件命名格式

- `1.mp4` - 第1张幻灯片的视频
- `1.png` - 第1张幻灯片的图片
- `2.mp4` - 第2张幻灯片的视频
- `2.jpg` - 第2张幻灯片的图片
- ...以此类推

## 🎯 使用说明

1. 将您的视频和图片文件按数字顺序命名
2. 放置在此目录中
3. 系统会自动识别并应用到对应的幻灯片
4. 视频优先级高于图片

## 📋 支持的格式

**视频**: mp4, webm, ogg
**图片**: jpg, jpeg, png, webp, gif

## 💡 优化建议

- 视频文件大小控制在10MB以内
- 图片文件大小控制在2MB以内
- 推荐分辨率1920x1080或更高
